# HR-Analytics-Project
HR Analytics uses data-driven methods to understand and improve workforce performance.
 >It helps organizations make informed decisions about hiring, retention, and employee satisfaction.
 >The project focuses on analyzing factors such as attrition rate, job satisfaction, income, and work-life balance.
 >Tools like Excel, Tableau, Power BI, and SQL are used for data analysis and visualization.
 >The aim is to identify patterns and insights that can reduce employee turnover and enhance productivity.
 >By applying HR Analytics, companies can create a strategic, efficient, and employee-friendly work environment.
